This is Task number 3 :
AI Image Generator which generates images using hugging face API based on user prompts and selection of given styles.

#Features
-Text prompt input
-Style options
-AI image Generation
-Prompt history
-Streamlit UI


#How to Run

1.Install

pip install -r requirements.txt

2.Run

streamlit run task3.py

#Project Structure

task3.py  Streamlit UI
image.py  Hugging Face API
s.py      Style selection prompt
requirements.txt
README.md
